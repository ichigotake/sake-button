# sake-button

[居酒屋選びで困ったときに押すボタン](https://sake-button-cbb77.firebaseapp.com)

## Getting started

### セットアップ

```bash
$ npm install -g firebase-tools
$ cd functions && npm install && cd ..
$ firebase login
$ firebase use --add
```

### サーバーを起動する

```bash
$ firebase serve
```

### Firebase へデプロイ

```bash
$ firebase deploy --project <projectId>
```
